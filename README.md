# Thought Process in Creating My Quarto Website

The initial setup of the terminal and then launching my website on GitHub Pages proved to be one of the most challenging aspects of following Sam’s tutorial. I ended up creating and subsequently deleting five different repositories on GitHub because my content wasn't uploading onto the site. However, after conducting some online research, troubleshooting, and closely studying Sam’s tutorial, I was able to learn from my mistakes and successfully launch my website on GitHub! It's difficult to pinpoint what aspects of creating the website were easy for me, as all the concepts of web development were new. However, repeating certain tasks, such as creating customized SASS themes, helped me grasp the purpose of each line of code and how it contributed to the structure of my website. On the other hand, comprehending CSS and HTML was an entirely new topic for me. I often found myself accidentally clicking the inspect button on websites without fully understanding its significance in web development. Although it was confusing initially to grasp the structure of these elements, through trial and error, I gained valuable hands-on experience and learned along the way!
